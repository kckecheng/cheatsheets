# Cheatsheets

Personal technical cheatsheets, organized for [Obsidian](https://obsidian.md).

Start at **[[MOC]]** — the Map of Content indexes every note by domain.

## Structure

- **Flat vault** — all notes in the root folder.
- 4 former monolith files (`git.md`, `linux.md`, `linux_net.md`, `troubleshooting.md`) were split into thematic atomic notes; smaller files (`ai`, `code_utils`, `go`, `misc`, `switch`, `vim`) are kept intact.
- Every note has YAML frontmatter: `tags`, `aliases`, `type`.
- Notes cross-link via `wiki-links`; see each note's *Related* section and Obsidian's Backlinks pane.

## Navigation

- Open [[MOC]] for the full index.
- Use the tag pane (`#git`, `#linux`, `#network`, `#debug`, `#cheatsheet`).
- Search by alias (e.g. "gdb", "iptables", "tcpdump", "cgroups").

## Using with Obsidian

1. Open this folder as a vault (*File → Open vault → Open folder as vault*).
2. Set [[MOC]] as your homepage (e.g. via the *Homepage* community plugin) for quick access.
3. Enable the **Tag pane**, **Backlinks**, and **Outgoing links** core plugins for full navigation.
